# Selected Projects

## BrightPath LMS
**React, Java, Spring Boot, JWT Authentication, PostgreSQL**

A full-stack Learning Management System inspired by platforms such as D2L. The system allows instructors to manage courses and students while providing a clean interface for students to join and interact with courses.

Key functionality includes:

- Secure JWT authentication
- Instructor and student role separation
- Course creation and enrollment
- Join code system for course access
- Protected API endpoints
- Production deployment with modern infrastructure

### Architecture

Frontend  
React application deployed on Netlify

Backend  
Java Spring Boot API hosted on Render

Database  
PostgreSQL with Flyway migrations

### Live Demo
https://brightpath-lms.netlify.app/

Demo accounts are available on the login page for quick testing.
